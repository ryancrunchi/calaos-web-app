Webapp for Calaos
-----------------

To build, install nodejs or yarn, and:
Then install dependencies:
With yarn:
```
yarn
```

With npm:
```
npm install
```

Then to build the app:
With yarn:
```
yarn build
```

With npm:
```
npx gulp
```

To develop with livereload:
With yarn:
```
yarn start
```
With npm:
```
gulp serve
```
Don't forget to set the ws://xxx/api URL in src/scripts/dev_config.js
